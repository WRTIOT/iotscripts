iotscripts
==========

script for iot platform

Python lib for IOT platform(www.lewei50.com)
	Thanks sanbinabu for subscribing

	1、先在电脑上安装python，我用的版本是python 2.7.6
	2、下载example.py和lewei.py两个文件，放在同一个目录下
	3、在乐联网上注册帐号，并添加名称为"01"的网关，两个名称为 "bat"和"temp"的传感器
	4、在example.py中填写你获得的Userkey，然后在命令行运行：python example.py。enjoy！
	5、可以试着玩玩TCP反向控制的接口，用来控制一些好玩的设备。
